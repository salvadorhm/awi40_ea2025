# awi40_ea2025

## 1. Crear un Virtual Enviroment

```shell
python3 -m venv .venv
```

## 2. Activar un virtual enviroment

```shell
source .venv/bin/activate
```

## 3. Desactivar un vitual enviroment

```shell
deactivate
```
